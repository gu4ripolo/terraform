# Terraform

Terraform scripts

## example-autoscale
It contains the scripts to create an Azure scale set. It installs nginx and stores the terraform state in Azure.

## example-kubernetes
It contains the scripts that create an Azure EKS.

## example-modules
It contains a script that creates a virtual machine using terraform modules.

## example-provisioner
It contains a script that uses terraform provisioners.

## example-vm
It contains a terraform script to create a Single Virtual Machine. It creates a resource group, virtal network and subnet, public ip and network security group and interface.
